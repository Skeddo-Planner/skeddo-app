## Notes & Outstanding Issues

1. **R46 warning on ID 15923** (Half Day PM, Ages 6-14): The 8-year age span is intentional — this is an aggregate listing covering PM sessions for ages 6-12 (Shooting Camp) and ages 8-14 (Goaltender Development) across the summer. The provider does break these into separate age groups per session, but the aggregate listing is retained as-is.

2. **Full-day Jul 27-31 split age groups**: The live page shows separate sessions for ages 6-8 and ages 9-12 for the Jul 27-31 week. Existing ID 328 covers only ages 6-12 (ageMax updated to 12). A separate per-age-group listing for this week was not added to avoid excessive granularity — the overlap with aggregate ID 15924 is noted.

3. **Half-day sessions not in database**: Several specific named half-day sessions (Hockey Fundamentals, Skate for Hockey, Shooting Camp, Power Skating Camp, Girls Hockey Camp AM, U9 Preseason, Battle Drills) are represented only through aggregate listings (IDs 15920-15923) rather than individual per-week entries. The aggregate approach is acceptable given the large number of sessions, but a future audit could break these out for better searchability.

4. **Learn to Skate programs**: The website references Learn to Skate (Parent & Child, Ages 3-6, Ages 7-12) at North Shore for Jan-March 2026. These sessions have already completed as of the audit date. No future 2026-27 skating lessons were found in the registration system.

5. **Skills & Drills waitlist sessions**: Tuesday sessions (both Beginner and Intermediate) for the 4-8 age group in Spring 2026 are Waitlisted. The aggregate listing ID 15926 shows "Open" because other day/time slots remain open. The costNote on 15926 should ideally mention the waitlisted Tuesday sessions — noted for a future update.
