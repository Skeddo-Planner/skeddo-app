# Skeddo — Program Search Methodology

## Purpose
This document defines the systematic approach for populating the Skeddo program directory when entering a new city. It ensures comprehensive coverage with minimal gaps, verified data, and consistent quality.

## Guiding Principles
1. **No shortcuts** — every program must be independently verified
2. **Registration URLs must point to actual registration pages**, not homepages
3. **Registration status must be verified** as of the current date
4. **Addresses must be real street addresses**, not city names
5. **Prices must be verified** from the provider's website (mark estimated if unsure)
6. **Age ranges must be accurate** — check the program page, not the provider's general info

---

## Phase 1: Municipal & Government Programs

### 1.1 City Parks & Recreation Department
- Find the city's recreation/parks department website
- Download or browse the summer program guide
- Check EVERY community centre in the city for:
  - Day camps (full day, half day)
  - Sports programs (swimming, tennis, basketball, etc.)
  - Arts programs (pottery, painting, drama)
  - Music programs
  - STEM programs
  - Special needs/inclusive programs
- Note registration dates — these often open on a specific date for all centres

### 1.2 School Districts
- Find all school districts serving the city
- Check each district's website for:
  - Summer learning programs (academic)
  - Summer enrichment programs (sports, arts, STEM)
  - Community education/community schools partnership programs
  - Spring break and Pro-D day camps

### 1.3 Libraries
- Check the city's library system for:
  - Summer reading clubs
  - STEM programs
  - After-school/summer drop-in programs

## Phase 9: Neighbourhood-by-Neighbourhood Sweep

For each neighbourhood in the city:
1. Search "kids programs [neighbourhood] [city] 2026"
2. Check the local community centre's full program listing
3. Search Google Maps for "kids" + "camp" + "class" within 2km radius
4. Look for small local providers (art studios, music teachers, dance schools)

## Data Quality Checklist (for every program added)

- [ ] **name** — official program name from provider website
- [ ] **provider** — full organization name
- [ ] **category** — matches one of: Sports, Arts, STEM, Outdoor, General, Multi-Activity, Performing Arts, Life Skills, Music, Academic, Faith-Based, Language, Cultural
- [ ] **activityType** — specific activity (e.g., "Soccer", "Coding", "Dance")
- [ ] **ageMin / ageMax** — verified from program page
- [ ] **cost** — verified from provider (number, in CAD)
- [ ] **startDate / endDate** — YYYY-MM-DD format, verified
- [ ] **days** — e.g., "Mon-Fri" (use day picker format)
- [ ] **times** — e.g., "9:00 AM - 3:00 PM"
- [ ] **address** — real street address with city and province
- [ ] **neighbourhood** — correct neighbourhood name
- [ ] **registrationUrl** — points to ACTUAL registration/program page (not homepage)
- [ ] **registrationDate** — when registration opens (if known)
- [ ] **enrollmentStatus** — verified as of current date:
  - "Open" — registration page is live and accepting
  - "Coming Soon" — confirmed reg date within 30 days
  - "Upcoming" — confirmed reg date 30+ days away
  - "Likely Coming Soon" — no confirmed date, estimated
  - "Full/Waitlist" — confirmed full
- [ ] **dayLength** — "Full Day" (6+ hrs), "Half Day" (< 6 hrs), or "Overnight"
- [ ] **season** — "Summer 2026", "Year-Round", etc.

---

## Registration Status Verification

**CRITICAL**: Do not trust static "Open" labels from data imports. Verify by:
1. Visiting the registrationUrl in a browser
2. Checking if the registration form/cart is actually functional
3. Looking for "Registration opens [date]" text on the provider's site
4. Checking the provider's social media for registration announcements

---

## Frequency of Updates

- **Weekly during registration season** (March-May): Re-check "Coming Soon" programs for status changes
- **Monthly during camp season** (June-August): Update Full/Waitlist statuses
- **Annually** (January-February): Full refresh for new year's programs
- **On user report**: When a user submits a program or reports incorrect data, verify immediately
